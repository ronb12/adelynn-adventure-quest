# Adelynn Adventure Quest

A Zelda-inspired 3D browser game built with React Three Fiber.

**Story: The Shattered Crown** — Adelynn collects 3 Crystal Shards to defeat Shadow Sorcerer Malgrath!

## Tech Stack
- React + Vite
- React Three Fiber / Three.js
- Zustand state management
- Tailwind CSS

## Play
